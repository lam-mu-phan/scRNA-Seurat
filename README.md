This repository contains my analysis of single-cell RNA sequencing (scRNA-seq) data from normal (NML) and fibrotic (IPF) human lungs (GSE132771).

I learned the Seurat workflow by following YouTube tutorial videos 1–18 (excluding videos 8 and 9):
https://youtube.com/playlist?list=PLOLdjuxsfI4N1SdaQQYXGoa5Z93hPxWVY

I summarized the key concepts and workflows in the scRNA_script.pdf document. This document includes standard Seurat command lines (in R) along with the corresponding plots generated from the analysis.

As this is my first time learning scRNA-seq analysis, more advanced methods and extensions will be explored and documented in future repositories.
